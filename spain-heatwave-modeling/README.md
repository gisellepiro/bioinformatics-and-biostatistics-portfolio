# Modeling the Geographic Extension of Heatwaves in Spain

## Overview

This project investigates the meteorological factors associated with the geographical extent of heatwaves in Spain between 1975 and 2024 using data from the Spanish Meteorological Agency (AEMET).

The objective is to identify which characteristics of heatwave events contribute most strongly to the number of provinces affected.

## Dataset

The analysis uses historical heatwave records published by AEMET covering the period 1975–2024.

Variables include:

- Year of occurrence
- Heatwave duration
- Temperature anomaly
- Maximum temperature reached
- Number of affected provinces

## Objectives

- Explore long-term trends in heatwave occurrence in Spain.
- Evaluate relationships between heatwave characteristics and geographical impact.
- Quantify the contribution of meteorological variables to heatwave extension.
- Assess model performance and predictive capacity.

## Methods

- Exploratory Data Analysis (EDA)
- Data visualization with ggplot2
- Feature engineering
- Multiple linear regression
- Model interpretation and evaluation

## Main Findings

- Heatwave duration and temperature anomaly were significant predictors of geographical extent.
- Maximum temperature alone showed limited explanatory power.
- The final model explained approximately 50% of the variability in the number of affected provinces.
